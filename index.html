Components of a Search Engine

Robot

A robot is a program that automatically traverses the hypertext Structure of the Web by retrieving a web page document and following the hyperlinks on the page. In addition, some robots may access and store the title, meta tag descriptions, and some text on the page.

Database

The collection of information that is organized so that it's contents can be accessed, managed, and updated. The web page that dispays the results of your search has information form the database that is used by he search engin site. Some search engines use some of the content recieved fomr other search engins. Such as Yahoo gets its primary search resilts from Google.

Search Form

The graphical user interface that allows a user to type in a word or phrase to search for. This is where user types in keywords related to their search. When submitted the keywords are sent to a script that searches the database using keywords. The results set is formatted with a link to each page, along with additional information that may include the page title, a breif description, the first few lines of text, or the size of the page. This varies by search engine.

Search Engine Optimization Tips

Key Words

Spend time brainstorming terms and phrases that people may use when seaching for your site. These terms are what describe your website or business.

Page Titles

A descrpitive page which includes your company name or website name will help your site market itself. The page title is often displayed in the SERP and saved when a visitor bookmarks your page. Do not use the exact same title for each page of your website, include keywords in the page title that are appropriot for the page.

Heading Tags

Use structures tags, h1, h2, etc. to organize your content. If appropriot include some keywords in the text contained with the heading tags. Some search engins will give higher list prosition if keywords are used in the page title or heading. However, do not spam keywords, meaning repeating the words over and over withing the content of the page. Search engins continue to become more sophisticated and spaming can prevent your page from being listed if it perceives you are trying to cheat the system.

Description

Write a few sentances about your website or business that makes it special and why people would want to visit. THe description should be inviting and interesting so that someone would choose your site.

Description Meta Tag

A self-contained tag that is placed in the head section of a web page and is dispalyed in the SERP by some search engins, such as Google.

Linking

SVerify htat all hyperlinks are working. Each page on your sebsite should be reachable by a text hyperlink and should be descriptive. Do not use phrases such as more info or click here. SHould use keywords as appropriate.

Images and Multimedia

Search engine robots do not see the text withing you images. Configure meaningful alt text for images and include relevant keywords.

Valid Code

While it is not required by search engins that your code be validated, well structred and valid code is more easliy proccessed by search engine robots which could help yur placement in the search engine.

Content of Value

Your website should contain high-quality, well-organized content that is of value to your visitors.

Addtional Techniques for th SEO

Map Your Site

An HTML site map os a web page with a map of the site that contains hierarchal list of hyperlinks to the major pages in your website. THis can be helpful for sebsite visitors and assists search engine robots as they follow hyperlinks on your site. An XML site mape is an XML file that is used by search engines only, not visitors. An XML site map provides the following information about each page to a search engine.

  • date that each page was last modified
  • indicator of how frequently each page changes
  • priority level for each page

You can use an online map generator to create a sitemap file.

Link Popularity

Search engines rate websites based on the number of sites that link to a particular sebsite and the quality of those sites. Link popularity can determine its order in the SERP.

SMO - Social Media Optimization

Make it easy for your current and potencial website visitors by creating content of value that is shareable. Such as providing links to Twitter, Facebook, and Pinterest.
